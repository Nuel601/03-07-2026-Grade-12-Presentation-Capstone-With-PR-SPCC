Quick Guide

Access Control Dashboard

Manage all user accounts, admins, and login page content. Data is stored in Firebase Realtime Database under 06_LogSystem.

  • User Online – number of active sessions across all pages.
  • System Status – shows "Online" when connected to Firebase.
  • Use the tabs to add/edit/delete accounts.
  • Changes are saved automatically to Firebase.
  • Passwords are never saved – they must be managed in Firebase Authentication or another secure system.
